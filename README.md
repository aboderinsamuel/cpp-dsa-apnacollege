# C++ Data Structures & Algorithms

A rigorously structured **C++ (C++17)** repository for mastering **Data Structures & Algorithms** with an emphasis on **clarity, performance, and interview-grade code**.

Designed to scale from fundamentals to advanced problem-solving while maintaining clean architecture and tooling discipline.

## Repository Structure

Each directory represents a core DSA domain.  
Every `.cpp` file is compiled as a **standalone executable** via CMake.
